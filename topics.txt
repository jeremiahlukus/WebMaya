iTerm2 - is what i use for a terminal (not a big deal )

1) basic navigation in terminal
    mkdir nameOfFolder - make directory (folder)
    cd  goHere - change dir
    ls - list contents of dir
    file path
    touch file.txt- creates a empty file
    rm file.txt   - removes a file
    rm -rf  folder  - removes everything in a folder

2) basic vim nav
    (i) insert mode
    (v) visual mode / (y) yank lines  (p) paste 
    (w) move to end of word
    ($) move to end of line
    (q) quit 
    (esc) exit mode
    (w) write 
    <Control- w>  new horizontal split 
    <C-w> s  split window horizontally
